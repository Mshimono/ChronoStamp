# ChronoStamp

This is TimeStamp GUI Application

## Prerequisites
- Rust
- cargo
- node
- tauri
- yarn

```sh
$ cargo -V
cargo 1.66.0 (d65d197ad 2022-11-15)
$ rustc -V
rustc 1.66.0 (69f9c33d7 2022-12-12)
$ node -v
v19.4.0
$ yarn -v
1.22.19
$ cargo-tauri -V
tauri-cli 1.2.1
```


## Usage

```sh
git clone git@github.com:Mshimono/chrno-stamp.git
cd chrno-stamp
yarn tauri build
```
